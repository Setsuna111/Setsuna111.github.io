(2020 PNAS) Neural networks to learn protein sequence-function relationships from deep mutational scanning data

发表于 2022-02-02 更新于 2022-02-25 分类于 Protein Design 阅读次数： Valine：
进行一个蛋白质设计论文的读

1.Introduction

  1. 监督学习方法，从深度扫描突变数据中心学习蛋白质序列与对应的功能的关系，并做了一系列实验&将该算法用于蛋白质设计（不还是数据驱动）
  2. 测试了多种不同的网络架构，最终表明捕捉非线性关系并考虑序列位置间信息共享的网络表现最好

2. Results

2.1 Deep Learning Framework

  1. 数据：使用深度扫描突变获得的蛋白质序列-功能数据
    这些深度扫描突变数据有一个蛋白序列扫描后变体具有的分数，表征功能或亲和力。注意并非和丙氨酸扫描突变一样，这个扫描突变可以生成更加丰富的变体，其反馈的分数应该和结合能有关（Wikipedia上指出是比较变体与野生型蛋白的活性以确定每个突变的影响，Enrich2文章中提出他给出了多重测量指标），但不一定等同于分数。公开可用的数据集一共五个，其中三个数据集在发布时就带有预先计算的功能分数，直接可以拿来用。其余两个运行Enrich2得到分数
    ENrich2：一款分析深度扫描突变数据的软件。输入扫描图便获得的多重指标即可
  2. 网络架构
    共测试了四种，关注一下他如何添加的三维结构信息：
    基于结构的图卷积网络，邻居定义：两个残基如果在一定三维距离之内则视为邻居
  3. 特征化编码残基序列的方法
    可以借鉴，在methods中有详细解释：
    使用one-hot编码氨基酸类别（20中可能的类别+1个终止密码子）
    AA-index编码氨基酸理化生物性质，有AAindex database 一共566中，通过主成分分析降到了19维（因为这566个是从多个研究中get的，有冗余）
    最后21+19拼接，再在序列维度上拼接
